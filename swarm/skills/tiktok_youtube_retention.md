# SHORT-FORM VIDEO ALGORITHM (YOUTUBE SHORTS / TIKTOK)

## 1. THE 3-SECOND RULE (RETENTION)
The algorithm only cares about one thing: Average View Duration (AVD). If they swipe away in the first 3 seconds, the video dies.
- **Visual Hook:** Sudden movement, text exploding on screen, weird imagery.
- **Audio Hook:** "Stop buying crypto in 2026 until you watch this."
- Cut all dead air. No breathing pauses. 

## 2. THE VISUAL PING-PONG METRIC
The viewer's eye must move to a new spot on the screen every 1.5 seconds.
- B-Roll change.
- Text color change (Yellow -> Red on important words).
- Sound effect (Swoosh, Pop) playing on key frames.

## 3. MONETIZATION FUNNEL
You cannot put clickable links in TikTok descriptions organically until you have 1k followers. 
- Use a "Link in Bio" CTA verbally at the end of the video.
- For YouTube Shorts, pin a comment with the affiliate link.

## 4. VISUAL CREATOR DIRECTIVE
When generating scripts, output a table format:
[ TIME ] | [ VISUAL ] | [ AUDIO/VOICEOVER ]
Ensure the script is no longer than 45 seconds when read aloud (approx 120 words). Specify exact pixel-art aesthetics or cyber-glitch effects for the editor to use.
